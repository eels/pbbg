## Getting Started Guide

### Clone

```bash
git clone https://github.com/eels/pbbg
```

### Install Yarn Dependencies

```bash
yarn install
```

### Build & Start Docker Resources

```bash
docker-compose build && docker-compose up --detach
```

### Start Development Server

```bash
yarn dev:web
```
